description: 简化代码以提高清晰度和可维护性——在不改变行为的前提下降低复杂度
---

调用 agent-skills:code-simplification 技能。

在保持行为完全不变的前提下，简化最近修改的代码（或指定范围）：

1. 阅读 CLAUDE.md 并学习项目约定
2. 确定目标代码——默认为最近的更改，除非指定了更广的范围
3. 在修改之前，理解代码的用途、调用者、边界情况和测试覆盖率
4. 扫描可简化的机会：
   - 深层嵌套 → 提前返回（guard clauses）或提取辅助函数
   - 过长函数 → 按职责拆分
   - 嵌套三元表达式 → if/else 或 switch
   - 含义模糊的命名 → 描述性命名
   - 重复逻辑 → 提取为共享函数
   - 死代码 → 确认后删除
5. 逐步应用每项简化——每次更改后运行测试
6. 验证所有测试通过、构建成功、diff 干净

如果简化后测试失败，回滚该更改并重新考虑。使用 `code-review-and-quality` 审查结果。
